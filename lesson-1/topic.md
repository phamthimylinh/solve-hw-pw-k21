## Tips
- Ổ "C" trên máy tính Windows thường để chứa hệ điều hành. Nếu hệ điều hành lỗi, toàn bộ dữ liệu trên ổ C dễ bị "bay màu". Do vậy, bạn nên sử dụng một ổ đĩa khác (ổ D, E, F) trên máy tính (nếu có) để lưu trữ code. Sẽ an toàn hơn nhiều ^^
- Hãy tạo một thư mục chứa tất cả các project của khoá học. Bạn sẽ tạo khoảng 3-5 project trong suốt quá trình học. Thật tuyệt vời nếu chúng được sắp xếp gọn gàng, ngăn nắp.

# Playwright
1. Tạo 1 folder code mới tên là `pw-course`, khởi tạo Playwright. Hãy thử đọc code và tư duy để sửa test 'has title' (nằm trong file tests/example.spec.ts) thành:  
    - a. Truy cập trang web: https://material.playwrightvn.com/
    - b. Kiểm tra rằng title contains text "Tài liệu học automation test"
2. Sửa text 'get started link':
    - a. Truy cập trang: https://material.playwrightvn.com/
    - b. Click vào link chứa text: "Bài học 1: Register Page"
    - c. Kiểm tra trang web có chứa thẻ heading với nội dung: "User Registration"
3. Tạo repo trên GitHub và push code lên.
4. Invite GitHub giảng viên vào repo (Giải viên khoá học của bạn. Nếu bạn học online, hãy invite Minh Phong)
